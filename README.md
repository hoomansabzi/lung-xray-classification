# Lung Disease Prediction from X-Ray Images

A deep learning project aimed at predicting and classifying lung diseases from chest X-ray images. This repository contains the code for training, evaluating, and testing models on the ChestMNIST dataset, part of the MedMNIST v2 benchmark.

## Project Overview
Automated analysis of chest X-rays can significantly aid radiologists in diagnosing common thorax diseases. This project builds a multi-label classification pipeline to detect various lung pathologies using lightweight, scalable deep learning models. 

## Dataset: ChestMNIST
This project utilizes the **ChestMNIST** dataset, which is a subset of the NIH-ChestXray14 dataset, standardized by the MedMNIST benchmark. 
* **Task:** Multi-label, binary classification (14 possible pathologies).
* **Image Modality:** Chest X-Ray (CXR).
* **Available Resolutions:** `64x64`, `128x128`, and `224x224`.

You can download the dataset and the associated MedMNIST code here:
* [MedMNIST Official Website & Code](https://medmnist.com/)
* [Zenodo Download Link (Dataset)](https://zenodo.org/records/10519652)
